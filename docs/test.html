The test function will run your unit tests in a sandboxed environment. It takes 3 parameters.

  • fileName is the file name of the runner
  • sandbox is an optional sandbox to use to run the tests in
  • cb is what's called after tests are completed
 
function test(fileName, sandbox, cb) {
  var file = path.join(process.cwd(), fileName);
  var exported = blackbox(file, sandbox);
  exported(cb);
}
